# 🧠 BAKOME HASH EA – Trend Following EA for MT5

**EMA crossover + RSI + ATR risk management, trailing stop, break-even, daily limits, filters.**  
[![MIT License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE) [![MQL5](https://img.shields.io/badge/MQL5-EA-005F99)](https://www.mql5.com)

## 🔥 Features
- Trend: EMA(50/200) on H1 · Momentum: RSI(14) >55/ <45  
- ATR‑based position sizing (RiskPercent) + dynamic SL/TP  
- Trailing stop, break‑even, daily loss/profit limits  
- Session (8‑20h), news, spread filters – order retry logic  
- No martingale / grid – MIT license

## 🛠️ Quick Start
1. Download `bakome_hash_ea.mq5` → place in `MQL5/Experts/`  
2. Compile (F7) → attach to **H1 chart**  
3. Adjust risk (RiskPercent = 1%) and filters as needed
